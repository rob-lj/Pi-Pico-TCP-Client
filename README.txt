Pico TCP Client
-------------------------------------------------------------------------------

This repository contains an example application targeting the Pi Pico W which
periodically connects to a TCP server (example Python server is also provided),
receives data, transmits data, and then disconnects.

The Raspberry Pi Pico SDK is required to build this project and is available at
https://github.com/raspberrypi/pico-sdk
